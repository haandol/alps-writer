<example>
### 7.1 기능 A (F1: 템플릿 기반 AI 문서 작성)

#### 7.1.1 사용자 스토리
- 사용자가 대화를 시작하면 AI가 ALPS 템플릿 기반으로 문서 작성을 안내
- 사용자는 AI의 안내에 따라 단계별로 문서를 완성

#### 7.1.2 UI 흐름
1. 앱 실행 시 환영 메시지와 함께 시작 안내
2. 사용자가 새 문서 작성을 원하면 AI가 첫 번째 챕터에 대한 질문 제시
3. 모든 챕터 완료 후 최종 문서 저장 안내

#### 7.1.3 기술 설명
- **시스템 프롬프트**: 템플릿을 통합하여 LLM에 제공
- **LLM 통합**: Amazon Bedrock의 Claude 3.5 Sonnet V2 모델 활용
</example>
